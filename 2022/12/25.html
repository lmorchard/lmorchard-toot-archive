@JohnShirley2023 @RUSirius Federated is collection of all the public posts your server hears about.

That's a subtle distinction from "everyone everywhere" because it depends on who your local server's users interact with. Or if the server gets public posts from a relay (many don't)

If no one on the local server interacts with anyone on some remote server, then posts from that server probably won't show up in Federated. Same goes for if the remote server is blocked from federation.
